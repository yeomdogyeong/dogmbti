# 🐶🧠 Dog-MBTI

 - 나와 맞는 성향의 강아지는 누굴까?
 - 배포링크 : https://dogmbti1031.netlify.app/

## 🌟 Features

- 퀴즈를 통해서 MBTI를 알아볼 수 있습니다.
- 결과에 따라서 나와 맞는 강아지를 찾아봅니다.
- 결과를 카카오톡 공유하기로 공유할 수 있습니다.
